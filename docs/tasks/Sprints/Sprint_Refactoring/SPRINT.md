## 1. Цель спринта

Полный рефакторинг UI/UX на основе утверждённого дизайн-брифа. Переход от текущего индиго-стиля (Tailwind defaults) к чистому премиум-минимализму с брендовой зелёной палитрой.

### Что меняется:
- **Палитра:** indigo → brand green (#22C55E)
- **Стиль:** generic → Farfetch-inspired clean minimalism
- **Компоненты:** обновление всех UI-компонентов под новые токены
- **Landing Page:** новая страница для неавторизованных пользователей
- **Все разделы:** каталог, авторизация, кабинет покупателя, кабинет поставщика

## 3. Задачи спринта

### TASK-R01: Landing Page
**Статус:** В работе
**Приоритет:** Критический

Внедрить Landing Page в React-проект. Страница открывается по главному домену (`/`) для неавторизованных пользователей. Авторизованные попадают сразу на каталог.

**Подзадачи:**
- [ ] Создать `LandingPage.tsx` на основе прототипа `landing.html`
- [ ] Обновить `tailwind.config.js` — заменить палитру indigo → brand green
- [ ] Обновить роутер: `/` → LandingPage (неавториз.) / CatalogPage (авториз.)
- [ ] Обновить Header — прозрачный вариант для лендинга, обычный для внутренних страниц
- [ ] Адаптивность: мобильная версия, бургер-меню
- [ ] Анимации: fade-in-up при скролле, hover-переходы

**Прототип:** [prototypes/landing.html](./prototypes/landing.html)
